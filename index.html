PROJECTS

Bus Attendance System (Group Project) - 2024

The Bus Attendance Monitoring System automates bus entry tracking in schools and colleges. A manual attendance system is prone to errors, leading to inaccurate records and inefficiencies. To overcome this, we developed a project that utilizes an RFID reader to track bus entry and exit times accurately.

Tech Stack: Raspberry Pi Pico, RFID Reader, RFID Scanner, ESP-32

Packet Sniffer Using Python (Individual Project) - 2024

A Python-based packet sniffer that captures and analyzes network traffic using the Scapy library. It logs packet details in real-time, saves them to a .pcap file for further analysis (e.g., in Wireshark), and supports filtering specific types of packets. This tool is useful for learning network protocols and debugging network issues.

Tech Stack: Python, TCP/UDP/ICMP Protocol

Smart Iv Drip Bag Monitoring System- 2025

Developed a smart IV bag monitoring system to detect fluid levels and alert medical staff in real time. Integrated sensor data with a microcontroller and implemented threshold-based alerting logic. Aligned with SDG 3 (Good Health and Well-Being) by enhancing patient safety and reducing manual monitoring.

Tech Stack: ESP,Load cell,hx711
